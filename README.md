Shape Management System là một ứng dụng Java console được xây dựng nhằm mô phỏng việc quản lý và tính toán các hình học cơ bản như Rectangle và Triangle.
Dự án tập trung vào việc áp dụng lập trình hướng đối tượng (OOP) kết hợp với mô hình MVC và Factory Pattern, đồng thời đảm bảo tuân thủ nguyên lý SOLID
